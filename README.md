## ローカルで確認する

リポジトリ直下でどれか一つ実行:

```bash
# Python 3（ログに http://127.0.0.1:8000/ が出る）
python3 -m http.server 8000 --bind 127.0.0.1
```

```bash
# Node.js (npx)
npx --yes serve -l 8000
```

```bash
# PHP
php -S 127.0.0.1:8000
```

ログのリンク、または http://127.0.0.1:8000/ を開く。日本語版は http://127.0.0.1:8000/ja/
