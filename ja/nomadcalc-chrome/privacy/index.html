プライバシーポリシー

NomadCalc Chrome 拡張機能 · 最終更新: 2026年6月25日

本プライバシーポリシーは、akifumi（吉村 明史）が開発する Chrome 拡張機能 NomadCalc が、利用時にどのような情報を扱うかを説明するものです。 NomadCalc は、ページ上で選択した価格に手数料・税・送料・カード手数料などを加味し、 実際に支払う金額の目安を確認するためのツールです。

概要

  • アカウント不要。 拡張機能内でログインは求めず、氏名やメールアドレスを収集しません。
  • 閲覧の監視はしません。 URL、ページ本文、Cookie、フォーム入力、閲覧履歴をバックグラウンドで読み取りません。
  • 選択テキストは明示的操作時のみ。 右クリックメニュー、キーボードショートカット、またはユーザーが開始したポップアップ／サイドパネルの操作時に限り、選択中のテキストを読み取ります。
  • ローカル優先の保存。 設定、レートキャッシュ、換算履歴は Chrome のストレージ API を使い、端末内に保存されます。
  • 広告・第三者分析なし。 広告 SDK は使用せず、データを販売しません。

拡張機能が処理する情報

選択したテキスト

換算を実行すると、アクティブなタブでユーザーが選択したテキスト（価格など）を読み取ります。 サイトによって通貨記号と金額が別要素に分かれている場合、DOM 上の近傍から通貨記号を補完することがあります。 選択テキストは通貨と金額の解析にのみ使われ、ページ全体のスナップショットとして収集・送信されることはありません。

設定と換算履歴

端末内（Chrome ストレージ）に保存されるデータの例:

  • 換算先通貨、言語、手数料・税・送料などの設定
  • 為替レートのキャッシュと取得時刻
  • 直近の換算履歴（最大50件：金額と通貨コード）
  • オンボーディングの選択内容、サイトごとの選択オーバーレイの有効化状態
  • 初回換算成功時刻やモバイルアプリ案内のクリック時刻など、端末内のみの製品指標

拡張機能を削除するか、chrome://extensions から拡張機能のデータを消去することで、 これらの情報を削除できます。

位置情報（任意）

位置情報の利用は任意です。許可した場合、現在地から現地通貨や近隣国の候補を提案します。 座標から国コードへの変換は端末上の組み込みルックアップで行います。 Nearby 機能利用時は、国コード・通貨コード・座標が Chrome ストレージにローカル保存される場合があります。 位置情報は当方のサーバーには送信されません。

為替レート API

最新レート取得のため、バックグラウンドの Service Worker から読み取り専用 API （nomadcalc-fx-rate-api.akifumi.workers.dev） へリクエストを送ります。公開のレート表の取得のみで、選択した価格・閲覧履歴・個人を特定する情報は含まれません。

任意のサイトアクセス（選択オーバーレイ）

初期設定時に全サイトへのアクセスを要求しません。ポップアップから特定サイトで選択オーバーレイを有効にした場合のみ、 そのサイト向けの任意ホスト権限を求めることがあります。許可するサイトは Chrome の拡張機能権限 UI で管理できます。

権限と利用目的

  • storage — 設定、レートキャッシュ、換算履歴を端末に保存
  • contextMenus — 右クリックで選択価格を換算
  • sidePanel — 詳細換算、手数料、履歴の表示
  • activeTab — ユーザー操作後に限り、選択テキストを読み取り
  • scripting — ショートカット実行時に選択読み取り用の小さなスクリプトを注入
  • offscreen — 任意の現地通貨検出のため、MV3 オフスクリーンドキュメントで位置情報を取得
  • FX API ホスト権限 — 読み取り専用の為替レート取得
  • 任意ホスト権限 — ユーザーが有効化したサイトでのみオーバーレイ表示

行わないこと

  • 許可したサイトを除き、閲覧中ページの改変は行いません（オーバーレイ UI の表示を除く）。
  • 個人情報の販売・貸与・取引は行いません。
  • 広告プロファイルの構築に拡張機能を利用しません。
  • 13歳未満から故意に情報を収集しません。

データの保持

ローカルに保存されたデータは、拡張機能のストレージを消去するかアンインストールするまで端末に残ります。 サーバー側の FX API に関するログ（存在する場合）も、公開レート配信に必要な運用データに限られ、 ユーザーの選択内容や閲覧行動は含まれません。

ポリシーの変更

拡張機能の変更に伴い、本ポリシーを更新することがあります。ページ上部の「最終更新」日付が最新版を示します。 更新後も拡張機能を利用し続ける場合、改定後のポリシーに同意したものとみなします。
